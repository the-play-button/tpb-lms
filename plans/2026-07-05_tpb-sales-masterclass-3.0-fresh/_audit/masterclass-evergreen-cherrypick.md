## Ce qui reste (candidats concrets, à valider un par un)

### A. La méthode pour apprendre une niche (source 6.1/6.2/6.3/6.4)
Le contenu SDR-agency est jeté. La **méthode** est réutilisable telle quelle pour le textile :
- se constituer un **lexique métier** pour parler comme un insider,
- lister les **pain points par catégorie** (revenue, talent, data, tech, client, croissance),
- repérer les **buying triggers via des signaux détectables** (job postings, avis Glassdoor,
  posts LinkedIn, changements de site, levées de fonds) + comment les détecter,
- les **objections propres à la niche**.
→ C'est le seul vrai gain net-new à mes yeux, et il sert directement le textile.

### B. La discipline CRM quotidienne du rep (source 4.4)
Habitude concrète : logger après chaque call, notes ≥ 2 phrases, follow-up posé sous 24h, jamais
de batch en fin de semaine.

### C. Le daily scorecard + ramp par phases (source 4.3)
Nouveau rep : suivre l'**activité** (pas les résultats) ; journée cadrée (prep / bloc d'appels /
clôture) ; phases Learning → Execution → Performance.
→ Attention : recoupe déjà en partie le Cours 6 (onboarding day-by-day) qu'on a shippé.

### D. Le feedback loop (source 5.3)
Le rep = antenne marché : remonter les objections / concurrents cités / features demandées, serré.
