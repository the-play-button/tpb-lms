## Les 8 cherry-picks evergreen (priorité décroissante)

### 1. 🏆 Closing toolkit — les techniques nommées (source: 3.6 Closing techniques)

Le plus gros net-new. Le LMS a un closing offre-spécifique (assumptive + guarantees + « stop
talking »). La masterclass ajoute une **boîte à outils de closes réutilisables** :

- **Signaux d'achat** : feux vert / orange / rouge (quand closer, quand creuser, quand reculer).
- **7 closes nommés** : ROI, Assumption, Urgency, Risk-Reversal, Alternative-Choice, Competitor,
  Scarcity (n'utiliser Scarcity que si vraie).
- **3 closes avancés** : Takeaway, Summary, Emotional.
- **La séquence** : trial close → traiter les concerns → réaffirmer la valeur → créer l'urgence
  → demander la commande → **se taire**.
- **Post-close** : oui → next steps immédiats ; non → follow-up gracieux (garde la porte).

→ **Cours 3 (Sales Conversation)**, nouvelle leçon « The Closing Toolkit ». Evergreen (les
scripts produit sont jetés, les techniques restent).

### 2. 🛡️ Objection-handling frameworks (source: 3.5 Objection handling)

Le LMS a 4 objections→guarantees (offre-spécifique, à jour). Net-new = les **méthodes**
transférables :

- **LAER** : Listen → Acknowledge → Explore → Respond.
- **Feel / Felt / Found**.
- **Boomerang** : retourner l'objection en raison d'acheter.
- **Réponse selon l'étape** : cold call = 30s max ; discovery = creuser la racine ; demo =
  montrer ; close = résumer la valeur + demander.
- **Prévention** : désamorcer les objections récurrentes AVANT qu'elles arrivent.

→ **Cours 3**, section craft de l'objection (méthode, pas l'offre).

### 3. 🔍 Discovery method générique (source: 3.3 Discovery playbook)

Le LMS a un discovery offre-spécifique excellent (les 4 choses à ramener, router le bucket…).
Net-new = la **méthode réutilisable** :

- **70/30** (écouter/parler), quantifier chaque chiffre, parler leur langue.
- **BANT+** : Budget / Authority / Need / Timeline / **Technical fit**.
- **Discipline de disqualification** : hard vs soft disqualifiers, disqualifier proprement > forcer.
- **Structure d'appel discovery** : ouverture → état actuel → douleur → fit → next step.

→ **Cours 3**, leçon « Discovery, the method » (méthode agnostique du produit).

### 4. 🎬 Demo method générique (source: 3.4 Demo Playbook + 2.2)

Net-new (le LMS n'a pas de leçon méthode-demo dédiée) :

- **Show-don't-tell** ; structure **Before → During → After → ROI**.
- **Customiser par persona** : décideur = ROI ; ops = process/data ; end-user = simplicité.
- **Traiter les objections PENDANT la demo** ; lire les signaux d'engagement.

→ **Cours 3**, leçon « The Demo », ou section de la leçon closing.

### 5. 🧭 What selling actually is — la fondation conceptuelle (source: 0. What is sales)

Net-new, evergreen, et vraiment actionnable comme cadre mental :

- **4 rôles** : Setters **SEEK**, Hunters **DEAL**, Farmers **CARE**, Supports **SAVE**.
- **Core Four** de l'acquisition : Outreach / Network / Content / Ads + **ambassadeurs**
  (customers / employees / agencies / influencers-partners).
- **« Many 1:1 »** = sales at scale (ni 1:many marketing, ni many:1 harcèlement).

→ **Cours 1 (Vision & Context)** ou ouverture du **Cours 3** : une leçon « What selling is »
qui pose le vocabulaire et le cadre.

### 6. 🎯 ICP profiling METHOD — réutilisable pour le textile (source: 6.1/6.2/6.3/6.4)

On **jette** le contenu SDR-agency, on **garde la méthode** de profilage d'une niche (directement
applicable aux fabricants de textiles plus tard) :

- Construire un **lexique métier** (parler comme un insider).
- Mapper les **pain points par catégorie** (revenue / talent / data / tech / client / growth /
  competitive).
- **Buying triggers = signaux détectables** + comment les détecter (job postings, Glassdoor,
  LinkedIn, website-change monitoring, funding DBs) + scoring urgence/douleur.
- **Objections sector-specific** (propres à la niche).

→ **Cours 4 (The Offer / Who We Sell To)**, leçon « How to learn a new niche » (méthode), qu'on
appliquera ensuite au textile.

### 7. 📊 Execution & reinforcement discipline (source: 4.3 / 4.4 / 4.1 / 5.1 / 5.2 / 5.3)

Le LMS a déjà roleplay/shadowing/kpis/onboarding (Cours 5 + 6). Net-new evergreen restant :

- **Activité, pas résultats, pour un nouveau rep** ; ramp par phases (Learning → Execution →
  Performance) + **daily scorecard** (morning prep / calling block / end-of-day).
- **CRM daily discipline** comme habitude rep : logger après chaque call, notes ≥ 2 phrases,
  follow-up posé sous 24h, jamais de batch en fin de semaine.
- **Weekly 1:1** (15 min : what's working / not / une seule skill à bosser).
- **Call library** : enregistrer les meilleurs calls, noter 3/4/5★, apprendre des wins.
- **Feedback loop** : le sales = antenne marché (capter objections / concurrents / feature
  requests et remonter serré).

→ **Cours 5 (Practice & Reinforcement)** : compléter là où c'est absent (CRM daily habit,
call library, weekly 1:1).

### 8. 💡 Principes / one-liners evergreen (source: 3.1 / 7)

- **Under-promise / over-deliver** (déjà dans c3-0 — garder).
- **Sell the vacation, not the plane** (déjà dans c1-3 — garder).
- **Toujours demander une referral après un close** (net-new, petit, à glisser dans le closing).
