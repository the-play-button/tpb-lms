## 2.0 — Entrée : Mailing List Reactivation (via CRM cleaning) — [AUTHORED, depuis l'offre Brutal CRM Truth]

**Ce qu'on vend (outcome)** : *« On réactive votre mailing list : on récupère le pipeline dormant,
on restaure votre délivrabilité, on vous rend une base sur laquelle vous pouvez réellement vendre.
Vos données restent les vôtres. Ça se rentabilise, ou on continue jusqu'à ce que ce soit le cas. »*

**Le mécanisme (à ne pas mettre en avant)** : le CRM cleaning — merges, reclassification lifecycle,
identification des dormants, flag des bounces, opt-outs, re-enrichissement. *(Doctrine Hormozi : on
promet l'outcome, on ne vend pas le mécanisme.)*

**Les 3 modules d'engagement** (le client configure selon son état) :
- **Assessment** — audit + diagnostic + scorecard + prescription (la **contrainte** qui bloque la croissance).
- **Execution** — le cleaning réel (TPB delivers).
- **Continuity** — maintenance continue (drift alerts, dedup au fil de l'eau).

**Segmentation client (3 buckets)** : Standard CRM (HubSpot/SF/Pipedrive → Assessment **free**,
lead magnet) · Standard + custom fields (**$5-15k**) · Custom CRM stack (multi-source / bridge /
post-merger → **$20-80k**). *Archetype Custom stack = Wonder Grip Europe (DECK + HubSpot).*

**Value ladder** : Entrée (Assessment) → Execution ($18k DIY → $70k DFY) → Continuity ($1-5k/mo) →
offres aval (LVL2/LVL3) selon le diagnostic.

> **[note anti-slop]** L'offre source contient du sur-ingénié à **écarter** en formation : le
> *Monthly Giveaway $602 800*, le *catalog 62 projets*, la combinatoire de garanties (WYMB Path
> A+B). On garde le cœur : 1 offre · 3 modules · diagnostic de contrainte · value ladder ·
> garanties simples (Never-A-Cost, Your-Data-Stays-Yours).

*[CARRY détaillé] product knowledge, ICP/triggers, discovery, objections, demo (scorecard),

**Ce qu'on vend** *(suppose LVL2 — a besoin de la data sales)* :

  le sales a validés/priorisés.

quel message résonne) que **seule l'activité sales génère**. ABM sans data sales = spray
firmographique (ce que font les lead agencies, ROI faible).
