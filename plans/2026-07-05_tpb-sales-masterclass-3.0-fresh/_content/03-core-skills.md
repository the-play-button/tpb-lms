## 3.1 — C.L.O.S.E.R (transversal) — [CARRY]

Le framework Hormozi adapté TPB (Clarify · Label · Overview · Sell the vacation · Explain concerns
· Reinforce), avec labels/scripts/ROI/objections/call-flow minuté. **Reste tel quel** — c'est LE
squelette de conversation, valable pour **chaque** offre. Doctrine **under-promise / over-deliver**
conservée.
**Enrichissement** : chaque offre a ses **labels de contrainte** propres (ex. Entrée = « votre
liste pourrit et vous ne le voyez pas » ; Outreach = « vos campagnes bouncent » ; Reactivation =
« vos reps perdent 50 % de leur temps en admin »).

## 3.5 — Objection handling — [CARRY]

Structure *rebuttal → pourquoi ils le disent → vraie inquiétude → follow-up → proof point*.
**Enrichissement** : banques d'objections **par offre** — Entrée (« on peut le faire en interne »,
« data ownership ») ; Outreach (« déjà trop d'outils », deliverability) ; Reactivation (« yet
another tool ») ; ABM (« budget marketing ≠ mon budget »).
